## 学习笔记

### 核心贡献

- 提出 HunyuanCustom，一个以主体身份一致性为核心、支持图像、音频、视频和文本四种模态条件输入的多模态定制化视频生成框架
- 基于 LLaVA 设计图文融合模块，实现身份信息与文本描述的深层交互，提升多模态理解能力
- 提出图像 ID 增强模块，沿时间轴拼接图像特征，利用视频模型的时间建模能力强化跨帧身份保留
- 针对音频与视频模态分别设计 AudioNet（层次化空间交叉注意力对齐）和 Patchify 视频驱动注入机制，实现多种条件解耦控制
- 在单主体和多主体场景上，ID 一致性、真实感和图文匹配度均显著超越现有开源与闭源方法

### 方法细节

- **基座模型**：基于 HunyuanVideo 视频生成框架构建，集成多模态条件注入能力
- **图像-文本融合模块**：引入 LLaVA 作为图文交互桥梁，使参考图中的身份信息有效融入文本描述，增强模型对「特定人物 + 特定场景描述」的组合语义理解
- **图像 ID 增强模块**：将参考图像特征沿时间维度拼接（temporal concatenation），使视频扩散模型在去噪过程中持续接收跨帧的身份参考信号，提升生成视频中主体外貌的稳定性
- **AudioNet 模块**：针对音频条件，通过空间交叉注意力（spatial cross-attention）实现音频特征与视觉特征的层次化对齐，使生成的人物唇形、面部动作与音频信号同步
- **视频驱动注入模块**：对条件视频进行潜在压缩，通过 Patchify 特征对齐网络将条件视频的结构与运动信息注入生成过程，支持视频驱动的物体替换或动作迁移
- **训练策略**：各条件模块与身份模块解耦训练，避免模态间冲突

### 关键发现

- 在单主体定制化场景中，ID 一致性和文本-视频对齐度（text-video alignment）显著优于 DreamBooth、IP-Adapter 等基线方法
- 多主体场景下模型能同时保留两个参考图中的身份，并在交互动作中保持各自特征不混淆
- 音频驱动定制化：模型支持给定人物图像 + 音频 + 文本描述，生成该人物在指定场景中说对应音频的视频，实现了灵活的音频驱动人物动画
- 视频驱动定制化：给定参考图像 + 条件视频，可将条件视频中的指定物体替换为参考图像中的主体，同时保留原视频的动作与环境
- 消融实验表明图像 ID 增强模块和 AudioNet 对身份一致性与音画同步均贡献显著

### 方法归类

- **所属范式**：扩散模型（Diffusion Model）+ 条件视频生成
- **技术路线**：基于 DiT（Diffusion Transformer）架构的多模态条件注入，条件特征通过交叉注意力与去噪网络交互
- **相关方法**：DreamBooth、IP-Adapter、Animate Anyone、HunyuanVideo
- **应用领域**：个性化视频生成、音频驱动人物动画、视频驱动物体替换、多模态内容创作
