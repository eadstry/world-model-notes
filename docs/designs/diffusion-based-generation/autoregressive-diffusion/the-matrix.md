## 核心思想

The Matrix 是首个能够生成连续 720p 高清真实场景视频流的基础世界模拟器，支持第一人称和第三人称视角的实时交互控制，并可在无剪辑的情况下生成长达一小时的连续视频序列。该工作的核心突破在于：利用 AAA 级游戏的有限监督数据（Forza Horizon 5、Cyberpunk 2077），结合大规模无监督真实世界视频（东京街景），实现了游戏环境到真实场景的零样本迁移。

The Matrix 在仅 16 FPS 的帧率下运行，但其视角一致性、场景多样性和零样本泛化能力远超同期工作。模型能够穿越沙漠、草地、水域和城市场景，甚至能模拟在训练数据中完全未出现过的场景——例如一辆宝马 X3 行驶在办公室环境中。这展示了 AAA 游戏数据在训练通用世界模型方面的巨大潜力，为弥合模拟与现实之间的鸿沟提供了新范式。

## 技术架构

### 1. 虚拟环境
混合数据源：
- **有监督游戏数据**: Forza Horizon 5（驾驶）、Cyberpunk 2077（城市探索），提供精确的相机位姿和动作标注
- **无监督真实数据**: 大规模的东京城市街景视频，无动作标注，用于提升真实世界视觉质量

### 2. 世界模型
- 基于扩散模型的自回归视频生成架构
- 条件包括：历史帧序列 + 相机控制信号（平移/旋转/速度）
- 核心挑战是在不同场景域（游戏↔现实）之间的平滑过渡和泛化
- 使用大规模预训练的视频扩散模型作为初始化，通过混合数据微调实现跨域迁移

### 3. 行动模型
- 第一人称和第三人称双视角支持
- 连续控制信号：6-DoF 相机运动（平移 + 旋转）+ 速度控制
- 控制信号通过条件注入模块融入到视频扩散的生成过程中

## 代码实现要点

暂无开源代码。

推测的技术要点（基于论文描述）：
- 使用预训练的视频扩散模型作为基础（可能基于类似 SVD 的架构）
- 在游戏数据上微调，然后通过在真实世界数据上继续训练实现跨域泛化
- 关键技巧：无监督真实世界数据与监督游戏数据的混合训练策略，利用 domain randomization 实现泛化

## 关键创新点

1. **混合数据训练**: 首次将 AAA 游戏数据和大规模真实世界视频数据结合，训练通用世界模型
2. **零样本泛化**: 模型能在训练数据中未出现的场景（如办公室驾驶）中生成逼真视频，展示强泛化能力
3. **超长时生成**: 可生成连续一小时的视频序列，远超同期工作（通常仅数分钟）
4. **双视角支持**: 同时支持第一人称和第三人称视角的实时交互
5. **AAA数据价值验证**: 证明了利用商用游戏数据训练世界模型的可行性——这在数据获取受限的场景下具有重要价值

## 代表性结果

- **分辨率**: 720p 高清视频流
- **帧率**: 16 FPS
- **生成时长**: 连续一小时无中断（未剪辑）
- **场景多样性**: 沙漠、草地、水域、城市街道等
- **零样本场景**: BMW X3 在办公室环境行驶（训练数据中不存在）
- **视角**: 第一人称和第三人称双模式
