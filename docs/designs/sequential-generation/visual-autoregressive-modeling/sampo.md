## 学习笔记

## 核心思想

SAMPO（Scale-wise Autoregression with Motion Prompt）提出了一种混合自回归世界模型框架，旨在解决当前自回归视频世界模型中的三个核心挑战：(1) **空间结构破坏**——传统的纯因果自回归逐 token 生成会破坏图像的空间局部性；(2) **解码效率低**——逐 token 的自回归生成速度慢，难以实现高效的长序列 rollout；(3) **运动建模不足**——缺乏对物体运动和机器人轨迹的显式建模，导致生成视频中动态区域不够真实。SAMPO 将**视觉自回归建模（VAR）用于帧内生成**与**因果建模用于帧间生成**相结合，实现空间效率和时序一致性的统一。

SAMPO 的三个关键设计：(1) **时序因果解码 + 双向空间注意力**——每个 token 可以看到当前帧所有已生成 token（双向注意力）和前序帧的所有 token（因果注意力），这保留了空间局部性并支持并行解码；(2) **非对称多尺度 tokenizer**——观测帧保留精细空间细节，未来帧使用更紧凑的动态表示；(3) **轨迹感知运动提示（Motion Prompt）**——注入物体和机器人轨迹的时空线索，将注意力聚焦于动态区域。SAMPO 在生成质量上达到竞争水平的同时实现了 4.4 倍推理加速。

## 技术架构

**Vision Encoding（视觉编码）**：SAMPO 的非对称多尺度 tokenizer 是核心亮点。对于观测帧（真实历史帧），使用高分辨率的多尺度 VQ-VAE，保留丰富的空间细节以提供准确的初始条件。对于未来帧（预测帧），使用较低分辨率的 tokenizer，提取更紧凑的动态表示——因为精确预测未来的纹理细节既不可行也不需要，关键是捕捉物体的运动趋势和场景的整体变化。这种非对称设计兼顾了初始条件的精确性和未来预测的效率。

**Knowledge Learning（知识学习）**：SAMPO 结合了两种生成范式：(1) **帧内生成**——使用视觉自回归（VAR）的方式，从低分辨率到高分辨率逐尺度生成当前帧的 token，每个尺度内利用双向注意力建模空间结构；(2) **帧间生成**——使用因果 Transformer，以历史帧 token 和运动提示为条件，自回归预测每帧的全局特征。时序因果解码 + 双向空间注意力的混合策略是 SAMPO 效率优势的关键——帧内可以并行解码，帧间保证时序因果性。运动提示模块以物体/机器人轨迹为输入，通过可学习的嵌入层将其映射为时空位置编码，在注意力计算中为动态区域赋予更高的权重。

**Controllable Simulation（可控模拟）**：SAMPO 通过动作条件模块接收控制信号（机器人关节角度或车辆方向盘/油门），将其注入到生成过程中。动作信号与运动提示模块协同工作——动作指定"什么操作"，运动提示指定"操作影响哪里"。这种双重控制在需要精确操作的任务中特别有效（如机器人抓取、车辆换道）。模型还展示了零样本泛化能力——在未见过的任务和环境上保持生成质量。

## 代码实现要点

核心架构：(1) 非对称多尺度 VQ-VAE tokenizer（高分辨率观测编码器 + 低分辨率未来编码器）；(2) 视觉自回归解码器（逐尺度生成 + 双向注意力）；(3) 因果 Transformer 帧间建模器；(4) 轨迹感知运动提示模块。推理加速度的关键在于帧内并行解码——在同时解码一个尺度内的所有 token 而非逐 token 生成。模型也展示了 scaling 行为——更大的模型参数量持续改善生成质量。

## 关键创新点

- **混合 VSAR + 因果范式**：首次将视觉自回归（帧内）和因果建模（帧间）融合为一个统一的 world model 框架
- **非对称多尺度 tokenizer**：观测帧与未来帧使用不同粒度的编码，实现了精确性与效率的 Pareto 最优
- **轨迹感知运动提示**：显式建模物体和机器人的运动轨迹，引导模型关注动态区域
- **4.4 倍推理加速**：通过帧内并行解码大幅提升 rollout 效率，同时保持竞争性的生成质量
- **零样本泛化与 scaling**：展示了在未见任务上的泛化能力和随模型规模线性改善的质量提升

## 代表性结果

在动作条件视频预测基准上，SAMPO 在 FVD、PSNR、SSIM 等指标上达到了与纯因果自回归方法竞争的精度。推理速度方面，4.4 倍加速使长序列 rollout 从分钟级缩短到秒级，使模型更适用于需要大量模拟的 MBRL 场景。在 model-based 控制任务中，基于 SAMPO 规划的策略在成功率和效率上均超越基于传统自回归世界模型的策略。运动提示的消融实验显示，移除运动提示后动态区域的预测质量（特别是快速移动的物体）显著下降。Scaling 实验验证了更大的模型在未见过的任务上表现更好。
