# Visual Autoregressive Modeling — 视觉自回归建模

::: tip 定义
Visual Autoregressive Modeling 是 Sequential Generation 范式中最纯粹的子路线。它不依赖预训练大语言模型，而是从零开始在视频数据上训练一个**端到端的自回归视觉世界模型**。输入为历史视频帧与动作 token，输出为逐帧生成的未来视频序列。
:::

## 核心思想与设计空间

视觉自回归建模的基本假设是：**世界可以通过「看过什么 → 做了什么事 → 接下来看到什么」的自回归映射来近似**。形式上，模型学习 $p(f_{t+1} | f_{0:t}, a_{0:t})$，其中 $f_i$ 为视频帧（或其 token 表示），$a_i$ 为动作信号。

这一范式的设计空间由三个关键维度刻画：

1. **视觉 tokenization 方案**：将连续帧映射到离散 token 空间还是连续嵌入空间？
2. **序列建模架构**：使用因果 Transformer 还是专门为视频设计的 Spatio-Temporal Transformer？
3. **动作条件机制**：动作信号通过何种架构注入生成过程？

这三个维度上的不同选择导致了该范式内部的高度多样性。

## 演进脉络：从 Genie 到 Genie 3

### Genie（2024, Google DeepMind）

Genie 是视觉自回归世界建模的里程碑式工作。其核心贡献在于证明了三个关键结论：
- 仅从 **2D 平台游戏视频**（无动作标签）中即可训练出一个可交互的世界模型
- 模型能自发地学习**隐式动作空间**——即无需显式动作标注，模型通过预测帧间变化自行发现了有意义的动作维度
- 生成的环境**可以接受用户输入并实时响应**，实现了从「看视频」到「玩游戏」的跨越

Genie 的技术架构使用了 ST-Transformer（Spatio-Temporal Transformer）作为骨干，通过 VQ-VAE 将视频帧编码为离散 token，使用可学习的 latent action 作为条件信号。其隐式动作学习机制尤为精妙：模型在训练时没有动作标签，但通过设计一个动作推理模块（从连续两帧推断 latent action）和生成模块（基于历史帧和推断的 action 生成下一帧），迫使 latent action 空间自然涌现出有物理意义的维度。

Genie 的局限性也很明显：只能处理 2D 横向卷轴游戏场景，无法泛化到 3D 环境；隐式动作空间不可解释，难以控制。

### iVideoGPT（2024）

iVideoGPT 将视觉自回归建模推广到**更广泛的交互式视频场景**（包括机器人操作、自动驾驶、游戏等）。其核心技术创新包括：

- **统一 token 空间**：将视觉观测、动作、奖励等多模态信号统一编码为离散 token，构建统一的 world token 序列。
- **Causal Transformer with Cross-Attention**：使用因果 transformer 进行自回归建模，但通过 cross-attention 层显式地注入动作条件——这比 Genie 的 additive embedding 更为强大。
- **Scalable Pretraining**：在多个领域的视频数据上进行联合预训练，探索了世界模型的多领域迁移能力。

iVideoGPT 的架构标志着视觉自回归建模从「特定领域小模型」向「通用基础模型」的转向。其统一 token 空间的设计为后续的 VideoWorld 和 Genie 3 奠定了基础。

### VideoWorld（2025）

VideoWorld 是视觉自回归 VWM 领域**目前最大的单一研究**，在 6.67 亿参数规模上进行了探索。其关键发现是：

- **模型在从未见过的游戏中展示了零样本交互能力**——模型仅在大量 2D 游戏视频上训练，却能在全新的游戏中作出合理的交互反应
- **知识迁移现象**：在一个游戏中学会的交互策略可以部分迁移到结构相似的另一游戏中
- **Scheduled Sampling 的精细调度**对于长程生成质量至关重要

VideoWorld 还提出了一个重要的方法论反思：当模型规模足够大时，纯视觉自回归能否绕过对显式物理建模的需求，直接从数据中涌现出物理理解？

### MineWorld（2025）

MineWorld 将该范式扩展到**3D 开放世界场景**（Minecraft），这是从 2D 到 3D 的关键跨越。其技术贡献包括：

- **3D-aware tokenization**：设计了能保留深度信息的视觉 tokenization 方案
- **层次化动作空间**：将复杂操作（建造、制作、探索）分解为层次化的动作 token
- **长程记忆机制**：通过引入 memory token 来缓解 3D 场景中的遮挡和视野外物体追踪问题

MineWorld 在 Minecraft 中实现了长达数分钟的连贯交互序列生成，这是此前视觉自回归模型从未达到的时长。

### Genie 3（2025）

Genie 3 是 Genie 系列的第三代，代表了当前视觉自回归建模的最高水平。核心飞跃包括：

- **从 2D 到 3D**：通过引入 3D-aware VAE 编码器，Genie 3 能够在 3D 场景中进行交互式生成
- **大规模训练**：在比 Genie 2 大数个数量级的数据集上训练（涵盖游戏、仿真、机器人等多种领域）
- **实时交互**：推理速度大幅优化，支持近实时的用户交互（<100ms 延迟）

Genie 3 的核心技术洞察是：**世界模型的能力几乎完全取决于数据多样性和模型规模**，架构创新反而是次要的。这一结论与语言模型的 scaling law 高度一致。

## 关键技术：视觉 Tokenization 的演进

视觉 tokenization 是该范式的第一道技术关口。其演进路径如下：

| 方案 | 压缩比 | 特点 | 代表工作 |
|------|--------|------|---------|
| **VQ-VAE** (2D) | 4×-16× | 逐帧编码，无时序压缩 | Genie 早期 |
| **VQ-VAE** (3D) | 4×4×4 (时空) | 同时压缩空间和时间 | MAGVIT-v2 |
| **ViT-VQGAN** | 16×-32× | Transformer-based，更高质量 | iVideoGPT |
| **MAGVIT-v2** | 4×8×8 | 因果 3D 卷积，专为视频设计 | VideoWorld |
| **Continuous VAE** | 8×-32× | 放弃离散化，连续隐空间 | Genie 3 |

当前的趋势是向**连续隐空间**方向收敛。离散 tokenization 的信息损失在需要精确物理交互的场景中是不可接受的。连续 VAE（如 SD-VAE）在保持压缩率的同时避免了量化误差，但也失去了与 LLM 基础设施直接兼容的优势。

## 关键挑战与研究方向

### 1. 长程生成的一致性与漂移

这是视觉自回归建模最核心的挑战。当前的对策包括：
- **Conditional Frame Injection**：每隔固定步数注入真实帧作为锚点（仅训练时可用）
- **Adversarial Training**：引入判别器迫使长期生成保持视觉一致性
- **Latent Planning**：在隐空间中进行贪心搜索或 beam search，选择未来最「合理」的轨迹

### 2. 动作空间的发现与构建

野生的互联网视频不包含动作标注。如何从纯视频中提取有意义的动作表征？三种策略：
- **隐式动作发现**（Genie 路线）：让模型自行学习 latent action
- **伪动作标注**（从字幕、光流、目标检测中推断动作信息）
- **强化学习微调**：在交互环境中通过 RL 赋予 latent action 明确的操作语义

### 3. 从 2D 到 3D 的跨越

2D 游戏视频中的世界建模相对简单（缺乏深度歧义、遮挡、3D 旋转等）。MineWorld 和 Genie 3 证明了 3D 视觉自回归建模是可行的，但生成质量距实用仍有较大差距。核心难点在于 3D 结构的 tokenization——既需要表示几何信息，又要保持 token 效率。

### 4. 实时交互性能

世界模型的交互性要求推理延迟在 100ms 以内。对于需要逐 token 解码数百步的自回归模型而言，这是一个巨大的工程挑战。当前的加速策略包括 KV-cache 复用、投机解码（speculative decoding）、以及将部分计算从 transformer 迁移到更高效的卷积模块。

## 总结与展望

Visual Autoregressive Modeling 代表了世界建模中最「纯粹」的一条路线：让模型从原始视频中自主学习世界的运行规律。Genie 系列从 2D 游戏到 3D 开放世界的演进，展示了 scaling 的力量——足够大的模型和足够多的数据，可以使世界理解自发涌现。但该范式的本质性局限（自回归误差累积、像素级生成的高昂代价）意味着，它更适合作为**视觉生成型 VWM**（如交互环境构建），而非**规划型 VWM**（如机器人控制）。未来的突破方向可能在于与 State Transition 范式的融合——用自回归生成处理视觉外观，用隐空间转移处理物理动力学。
