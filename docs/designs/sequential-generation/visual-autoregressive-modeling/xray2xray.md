## 学习笔记

## 核心思想

Xray2Xray 提出了一种新颖的世界模型范式——从 2D 胸部 X 光片（CXR）中学习编码 3D 体积结构信息的潜在表示。胸部 X 光片是全球使用最广泛的医学成像模态，但作为 2D 投影图像，它们受到结构叠加（structural superposition）的根本性限制——三维解剖结构在二维平面上重叠，导致关键病变可能被遮挡或误判。Xray2Xray 的核心洞察是：通过建模不同角度 X 射线投影之间的转换动力学，可以让模型学会隐式地建模 3D 胸部体积结构。

这本质上是一种**视角转换型世界模型**——模型不一定需要真实的多视角数据，而是学习了"如果我有一个 3D 体积理解，那么不同角度的 X 射线应该满足什么样的投影关系"。这种从 2D 投影序列中提取 3D 信息的能力，使 Xray2Xray 的潜在表示在心血管疾病风险评估和病理分类等下游任务中超越了监督学习和自监督预训练方法。此外，论文还展示了从潜在表示重建体积上下文的能力。

## 技术架构

**Vision Encoding（视觉编码）**：Xray2Xray 使用专用的医学影像编码器（CNN backbone）将 2D X 射线图像编码为潜在表示。编码器需要学会从单张 2D 投影中提取足够的信息，使后续的过渡模型能够推断出 3D 体积结构。由于医学影像的高分辨率和精细纹理要求，编码器在架构上注重保留高频细节和器官边界的结构信息。

**Knowledge Learning（知识学习）**：核心包含两个子模型：(1) **视觉模型**——将 CXR 编码为潜在表示；(2) **过渡模型（Transition Model）**——建模不同角度 X 射线投影之间的演变关系。过渡模型实质上是一个潜在空间中的"视角旋转器"——给定当前角度的潜在表示，预测旋转到新角度后的潜在表示。通过这种视角转换自监督任务，过渡模型被迫学习胸部解剖结构的 3D 先验知识（如心脏在左侧、肺叶的形状等）。额外使用 CT 重建的伪多视角数据进行监督，进一步提升 3D 理解的精度。

**Controllable Simulation（可控模拟）**：Xray2Xray 的"可控性"体现在两个层面：(1) **视角控制**——用户可以指定目标投影角度，模型生成对应角度的虚拟 X 光片；(2) **潜在空间操作**——下游任务可以直接使用潜在表示进行分类/回归，而不需要显式重建多视角图像。这种潜在空间操作相比在图像空间操作具有更高的计算效率和信息密度。

## 代码实现要点

视觉模型基于 ResNet/DenseNet 架构的 CNN 编码器，输出一个紧凑的潜在向量。过渡模型使用 MLP 或轻量 Transformer，以当前潜在向量 + 目标角度编码为输入，预测新角度的潜在向量。训练使用 NT-Xent 对比损失（对齐同一患者不同角度 CXR 的潜在表示）和均方误差（监督 CT 重建的伪多视角投影）。评估指标包括心血管疾病风险的 AUC 和 5 种病理的分类准确率。CT 重建伪多视角数据作为额外的监督信号用于 fine-tune。

## 关键创新点

- **首个 CXR 世界模型**：将世界模型的视角转换范式引入医学影像领域，从 2D 投影中提取 3D 体积理解
- **隐式 3D 推理**：不需要 CT 数据作为输入，仅通过建模多角度投影的过渡关系来学习 3D 结构
- **超越监督学习**：在下游疾病预测任务中，学到的潜在表示比标准监督学习和自监督预训练更具判别力
- **体积上下文重建**：演示了从潜在表示合成新角度 CXR 的能力，为放射科医生提供多视角辅助诊断

## 代表性结果

在心血管疾病风险评估任务上，Xray2Xray 的潜在表示超越了多个监督方法和自监督预训练基线（如 SimCLR、BYOL），AUC 提升约 2-3%。在 5 种病理分类任务中，达到了与有监督方法竞争的性能。通过潜在表示重建的多视角 X 射线在视觉质量和解剖结构准确性上优于显式的新视角合成方法。消融实验确认了视角过渡模型对 3D 理解的关键贡献——移除过渡模型后，下游任务性能显著下降。
