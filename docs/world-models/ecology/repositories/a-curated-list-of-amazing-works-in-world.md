## 学习笔记

### 核心思想

该 GitHub 仓库由社区维护，旨在系统性地整理世界模型领域的高质量学术论文和工程资源。其范围覆盖了具身智能、自动驾驶、自然语言处理等多个应用领域，是目前社区中最全面、维护最活跃的世界模型 Awesome 列表之一。该仓库以分类清晰、更新及时著称，为研究者提供了从入门到深入研究的完整论文导航体系。

该资源列表采用多层级分类结构，将世界模型相关工作按照模型架构（如扩散模型、自回归模型、JEPA）、应用领域（具身智能、自动驾驶、NLP）和评测基准等维度进行组织，便于研究者根据自身需求快速定位目标文献。此外，仓库中许多条目附带了论文链接、代码链接和简要说明，极大地降低了研究者的文献筛选成本。
