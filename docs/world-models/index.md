# 世界模型 (World Models)

世界模型研究旨在让机器具备对物理世界的感知、预测与推理能力。本目录按研究维度展开：

## 子分类

| 维度 | 说明 | 入口 |
|------|------|------|
| 理论基础 (Fundamentals) | 世界模型的本质、物理可解释性、评测方法论 | [→](./fundamentals/) |
| 架构设计 (Architectures) | 四大核心范式：Sequential / Diffusion / Embedding / State Transition | [→](./architectures/) |
| 数据集与评测 (Datasets & Benchmarks) | 通用预测、物理因果推理等评测体系 | [→](./datasets-benchmarks/) |
| 下游应用 (Applications) | 具身智能、自动驾驶、游戏等实际场景 | [→](./applications/) |
| 研究生态 (Ecology) | 综述、Workshop、开源仓库、跨学科视角 | [→](./ecology/) |
