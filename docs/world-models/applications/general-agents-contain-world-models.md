# General agents contain world models

::: info 论文信息
- **论文全称**: General agents contain world models
- **arXiv**: [https://arxiv.org/abs/2506.01622](https://arxiv.org/abs/2506.01622)
:::

## 学习笔记

### 核心思想

该工作提出了一个深刻且具有范式意义的观点：通用智能体天然包含世界模型。作者通过理论分析和实验证据表明，任何能够在复杂环境中做出有效决策的通用智能体，其内部必然隐式或显式地构建了某种形式的世界模型。
该工作的意义在于为世界模型研究提供了一种新的动机框架——世界模型不仅是可选的辅助组件，而是通用智能的必然组成部分，这从根本上提升了世界模型研究在通用人工智能路线图中的战略地位。
