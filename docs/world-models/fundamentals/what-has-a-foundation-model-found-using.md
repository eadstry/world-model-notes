## 学习笔记

### 核心思想

该工作提出了一个巧妙的方法论：利用归纳偏置探针（Inductive Bias Probes）来探测基础模型内部是否形成了世界模型。通过设计一些特殊的测试任务，这些任务的成功完成仅取决于模型是否"理解"了相关的物理或逻辑规则，作者得以窥探基础模型的内部世界表征结构。
该工作的创新之处在于其探索方法不依赖任何关于模型应该学习什么的先验假设，而是通过系统性的控制实验来反向推断模型的内部知识结构。
