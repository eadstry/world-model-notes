# Embedding Prediction（嵌入预测范式）

::: tip 核心哲学
Embedding Prediction 范式的核心主张是的*世界模型不必生成像素，只需在隐空间中预测世界的表示变化的* 其代表性方法论 JEPA（Joint Embedding Predictive Architecture）将「从视觉观测中学习世界模型」定义为「预测被掩蔽的未来观测在嵌入空间中的表示」，而非「逐像素重建被掩蔽的区域」的:::

## 范式哲学：为什么不生成像素。
Embedding Prediction 范式的设计理念源于对生成的VWM 的深刻反思。生成式世界模型（Sequential Generation 的Diffusion-based）存在一个结构性浪费：**模型将大量容量用于生成像素级细节（纹理、光照、背景），而这些细节对世界状态的理解和决策几乎没有贡献*。
以自动驾驶为例：理解「前方车辆正在变道」这一世界状态变化，只需要一个紧凑的语义表示（位置、速度、意图），而无需生成该车辆的一切像素细节（车牌上的污渍、车窗的反光、车漆的颜色渐变）。生成式模型把大量计算浪费在了这些无关细节上。
JEPA 的核心洞察是统*一个好的世界表示应该最大化对世界状态的表达能力，同时最小化对像素细节的编码需要*。这意味着预测应该在学习的嵌入空间中进行，而非像素空间。
## 从 JEPA 到 V-JEPA：方法论建立

### JEPA 的基本原。
JEPA（Joint Embedding Predictive Architecture）最初由 LeCun 提出，作为通向人类水平 AI 的核心路径之一。其基本架构包含三个组件。
1. **Context Encoder** $E_c$：将可观测的上下文$x$ 编码为上下文表示 $s_x$
2. **Target Encoder** $E_t$：将需要预测的目标 $y$ 编码为目标表性$s_y$
3. **Predictor** $P$：根据上下文表示 $s_x$ 和可选的潜变化$z$，预测目标表性$\hat{s}_y$

训练目标是最小化预测误差的\mathcal{L} = \| \hat{s}_y - s_y \|^2$（或其他距离度量。
JEPA 与生成式方法的本质区别在于：**预测在表示空间中进行，无需将预测结果解码回观测空间来验的*。这避免了生成式模型中最昂贵、最困难的步骤。
### V-JEPAt024, Meta。
V-JEPA 心JEPA 思想在视频领域的大规模实现。其设计要点。
- **Video Masking Strategy**：在时空维度上对视频块进行掩蔽——不是简单的随机掩蔽，而是时间上偏后（模拟「预测未来」）的遮挡模型- **Vision Transformer Backbone**：使用ViT 作为编码器骨干，将视频分割为 3D patches
- **特征空间预测**：Context encoder 处理可见的时空区域，Predictor 预测被掩蔽区域的表示

V-JEPA 的核心发现是潜*在嵌入空间中做预测，模型能自发学习到高层次的时空结构**——包括物体的运动模式、场景的几何结构、以及一些初级的物理规律（如物体不会凭空消失）。
### V-JEPA 2 2025。
V-JEPA 2 是第二代的改进版，核心变化包括：
- **更大的规模*：更大的模型、更多的数据、更长的视频
- **层次化预测*：不仅预测最终的嵌入，还预测中间层的特征，形成多层次的表示学习- **动作感知的掩码*：掩蔽策略开始考虑视频中的运动信息——运动剧烈区域的掩蔽概率更高

V-JEPA 2 最为人瞩目的发现是**特征预测能力的涌现*：当模型规模和训练数据达到某个阈值后，模型开始展现出对物理规律（如重力作用下物体的运动轨迹）的隐式理解，尽管从未被显式地教授过物理。
## JEPA 作为世界模型的优。
### 1. 计算效率的质。
JEPA 不需要生成像素，因此训练和推理的计算量比生成的VWM 一1-2 个数量级。这使得 JEPA 可以在更多数据上训练更大的模型，或在受限的计算预算下实现更强的世界理解。
具体的效率对比（以处理16 理256×256 视频为例）：

| 操作 | 生成的VWM | JEPA |
|------|-----------|------|
| 编码 | O(n²) ViT 编码 | O(n²) ViT 编码（相同） |
| 预测 | O(n²) 解码生成 | O(m²) 嵌入预测（m 的n试|
| 解码 | O(n²) 像素重建 | **无需解码** |
| 总推物| 理| 低（~10× 加速） |

### 2. 避免生成式模型的信息瓶颈

生成式模型必须通过一个像素解码器将隐空间表示映射回像素空间——这个解码过程是信息瓶颈，因为隐空间必须保留足够的像素级信息才能实现重建。JEPA 没有像素解码器，隐空间可以专注于编码**世界状态信息*而非**视觉细节信息**。这意味着在同等模型容量下，JEPA 的表示可以包含更丰富的语义信息。
### 3. 对下游任务的自然适配

JEPA 学习到的嵌入表示是多功能的——既可用于世界状态预测，也可直接作为强化学习的state representation、规划的 cost function、或迁移学习的通用特征。DINO-WM 的工作证明了 JEPA 表示在视觉规划任务上的优越性——在 VPT（Video PreTraining）的 Minecraft 规划任务上，JEPA 表示在样本效率上显著优于像素空间方法。
## JEPA 家族的工作演。
### DINO-WMa024-2025。
DINO-WM 是将 JEPA 嵌入预测的*世界模型规划**结合的关键工作。其架构。
- **Pre-trained Visual Encoder**：使用DINOv2（自监督预训练的视觉 transformer）作为冻结的特征提取潜- **JEPA-style Dynamics**：在 DINOv2 的特征空间中学习状态转交- **Goal-Conditioned Planning**：通过 Model Predictive Control（MPC）在隐空间中进行目标条件规划

DINO-WM 的核心贡献是**解耦视觉感知和世界动态*——使用现成的预训练视觉模型（的DINOv2）提供强大的视觉表示，而无需在世界模型训练中重新学习视觉编码。这种解耦使得世界模型的训练变得极其高效（数小时而非数天）。
DINO-WM 在以下任务上展示供strong performance了- **Minecraft 中的长程任务**（如「收集木头并制作工具」）
- **机器人操作规模*（从视觉观测中规划操作序列）
- **导航任务**（在隐空间中规划最短路径）

### seq-JEPAt025。
seq-JEPA 代表性JEPA 家族的*序列化世界建模*方向上的最新进展：

- **Sequential Prediction**：不同V-JEPA 一次性预测被掩蔽的所有未来块，seq-JEPA 采用顺序预测——先预测最近的未来块，再基于预测结果预测更远的未来。这更接近世界模型的「滚动预测」范式的- **Dynamic Masking**：掩蔽策略在训练过程中动态变化，从「预测短期未来」逐步过渡到「预测长期未来的- **Memory Module**：引入外部记忆模块存储长期历史信息，支持远超训练窗口长度的世界推。
seq-JEPA 的实验结果表明，顺序预测相比一次性掩蔽预测能更好地捕捉*因果结构**——因为顺序预测迫使模型关心事件之间的时序依赖关系，而一次性掩蔽允许模型通过「空间模式匹配」作弊（而非真正学习动力学）。
## 与其他范式的互补与融。
### JEPA + State Transition

DINO-WM 是该融合最成功的实例。JEPA 提供强大的视觉表示，State Transition 模型在该表示空间中学习紧凑的动力学。二者的结合产生了一个*视觉上强大、物理上合理、计算上高效**的世界模型。
### JEPA + Diffusion

一个新兴方向是一JEPA 的嵌入预测来**指导**扩散生成。例如，先让 JEPA 在隐空间中预测未来世界的「结构蓝图」（物体位置、运动趋势），然后将该蓝图作为扩散模型的条件信号。这可以显著提升扩散 VWM 的物理一致性。
### JEPA + Sequential Generation

JEPA 也可用于**压缩自回归序列*。代替在 token 空间中做 AR 预测，可以用 JEPA 在紧凑的嵌入空间中预测，仅在需要时解码图token/像素空间。这种设计保留了 AR 的时序灵活性，但将计算成本降低了数量级。
## 局限性与开放问。
1. **缺乏视觉输出**：JEPA 不生成图像，因此在需要可解释性视觉输出（如向人类展示「世界会变成什么样」）的场景中，必须额外配备一个解码器。
2. **预测精度的天花板**：在嵌入空间中做预测意味着模型无法感知低于嵌入粒度的视觉变化。对于需要精确视觉细节的任务（如精细操作），JEPA 可能力有不逮。
3. **嵌入空间的不变性假假*：JEPA 假设同一个嵌入空间可以容纳所有未来状态。但对于分布外（OOD）的观测，嵌入空间可能不再有效，导致预测崩溃。
4. **评估困难**：不像生成式模型可以通过「生成的视频是否真实」来直观评估，JEPA 的预测质量难以直接评估——通常只能通过下游任务来间接衡量。
## 总结

Embedding Prediction 范式代表性VWM 设计中的一种「少即是多」哲学。通过放弃像素生成，JEPA 家族在计算效率和表示质量上取得了显著的帕累托改进。DINO-WM 证明了JEPA 表示可以无缝衔接世界模型规划，seq-JEPA 展示了顺序预测在捕捉因果结构上的优势。尽管缺乏视觉输出的能力限制了其部分应用场景，但 JEPA 作为视觉理解和世界推理的「后盾」具有巨大的潜力——尤其是在与其他范式（State Transition、Diffusion）的融合中。
