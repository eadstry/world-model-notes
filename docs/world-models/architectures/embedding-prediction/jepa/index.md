# JEPA Family（联合嵌入预测架构家族）

::: tip 家族定义
JEPA（Joint Embedding Predictive Architecture）家族是 Embedding Prediction 范式的核心载体。其核心理念是Yann LeCun 从2022 年的「通向自主 AI 之路」纲领性论文中系统阐述多*智能的本质不在于生成像素，而在于预测世界的抽象表示的* 视觉世界模型语境下的 JEPA 家族——V-JEPA 分V-JEPA 2 的DINO-WM 分seq-JEPA——代表了这一理念从理论到实践的系统化落地图:::

## JEPA 的哲学基础：LeCun 的自主 AI 架构

在LeCun 的构想中，一个完整的自主 AI 系统应包含六个模块，其中**世界模型（World Model的*是最核心的组件。LeCun 认为，当前生成式 AI（如 LLM、扩散模型）的一个根本错误是试图用*原始观测空间**中进行预测和生成——这在低维空间中尚可工作，但在高维连续感知（如视觉）空间中必然导致指数级的容量浪费。
JEPA 的解决方案是一*将预测从观测空间迁移到表示空间*。具体而言。
1. 用编码器将高维观测$x$ 映射到低维表性$s = \text{Enc}(x)$
2. 在表示空间中做预测：$\hat{s}_{t+1} = \text{Pred}(s_t, a_t)$
3. 通过最小化 $\| \hat{s}_{t+1} - \text{Enc}(x_{t+1}) \|$ 来训练预测器

这与生成式方法的本质区别在于**能量函数的计算位置*——JEPA 在表示空间中计算损失，避免了将隐状态解码回像素空间的过程。
## V-JEPA：视频JEPA 的开创性实。
### 架构设计

V-JEPAt024）是 Meta 一JEPA 路线上的第一个大规模视频实现。其架构由三个核心组件构成：

- **Context Encoder**：Vision Transformer（ViT），输入为可见的视频 3D patches（未被掩蔽的时空块），输出为上下文嵌入序列- **Target Encoder**：相同架构的 ViT，输入为被掩蔽的 target patches，输出为 ground-truth target 嵌入
- **Predictor**：轻量Transformer，以 context 嵌入作mask token 为输入，预测 target 嵌入

其中 Target Encoder 的权重是 Context Encoder 的指数移动平均（EMA），这一设计避免了表示坍塌（representation collapse）——即模型将所有预测都坍缩为同一向量的退化解。
### 掩蔽策略设计

V-JEPA 的掩蔽策略是其最关键的设计选择同- **时序后偏**作5% 的掩码patch 位于视频的时间后半段，模拟「预测未来模- **块状掩蔽**：以连续时空块（而非随机散点）为单位进行掩蔽，迫使模型学习更高层次的时空结构
- **多尺度掩蔽率**：在训练中随机变化掩蔽率，使模型同时适应短期和长期预。
### 关键发现

V-JEPA 实验揭示了一个重要现象：**在嵌入空间中做预测，模型学会了编码「不变的高层次结构」而非「多变的低层次纹理解*。例如，模型对「一个移动的球」的嵌入预测是稳定的，不随光照、视角、背景的细微变化而大幅波动。这种不变性是世界模型作为下游任务基础的关键属性。
## V-JEPA 2：规模化与层次化

V-JEPA 2 2025）的核心贡献是在 V-JEPA 的基础上证明了两条 scaling 定律。
1. **模型规模 scaling**：随着 ViT 参数据300M 增加速2B，特征预测质量（在下游任务上衡量）持续提升，未见平台的2. **数据规模 scaling**：在更大规模、更多样化的视频数据上训练，模型展现出更强的跨领域泛化能。
V-JEPA 2 引入高*层次化预测*是另一重要创新：Predictor 不仅预测 target encoder 的最终输出，还预测中间层的特征。这形成了一个多层次的监督信号，迫使模型在不同抽象级别上都学会预测世界变化——低层预测纹理和运动，中层预测物体形状和关系，高层预测场景语义和事件结构。
**动作感知掩蔽**的V-JEPA 2 的另一创新：视频中运动剧烈的区域（高光流）被掩蔽的概率更高。直觉是：运动区域包含更丰富的物理状态变化信息，模型应该更多地从学习预测运动区域中获益。实验结果支持了这一假设。
## DINO-WM：JEPA 与世界模型规划的桥梁

DINO-WMa024-2025）是 JEPA 家族的*最直接面向世界模型应用的工作*。它的设计理念是将*复用现有的自监督视觉表示（DINOv2），在其上构建轻量的世界动力学模型*。
DINO-WM 的架构：
- **冻结构DINOv2 Encoder**：将视觉观测编码为强大的预训练特征表性- **JEPA-style Dynamics Predictor**：在 DINOv2 的特征空间中学习状态转交- **MPC Planner**：通过 Model Predictive Control 在特征空间中搜索最优动作序。
DINO-WM 的关键突破在于*视觉感知与动力学建模的彻底解的*。传统世界模型需要同时学习视觉编码和动力学——这意味着世界模型的容量被分成了两半。DINO-WM 利用 DINOv2（在数十亿张图像上预训练）的极致视觉表示能力，将全部容量集中在动力学学习上。
实验结果令人印象深刻的- 关Minecraft 的VPT 基准测试中，DINO-WM 的规划效率显著优于在像素空间中规划的方法
- 在机器人操作任务中，仅需数小时的训练即可学习可用的动力学模型
- 展现出对视觉干扰（光照变化、纹理替换）的强鲁棒性——因果DINOv2 的表示本身具有较强的不变。
DINO-WM 也暴露了 JEPA 路线的一个局限：**其动力学预测的质量高度依赖于预训练视觉表示的质量**。如何DINOv2 对某类场景的表示不够好（如非常规视角、特种领域），DINO-WM 的规划能力也会相应下降。
## seq-JEPA：顺序预测的因果性优。
seq-JEPAt025）对 V-JEPA 的一次性掩蔽预测范式进行了批判性反思。核心论点是统*一次性预测所有被掩蔽的未来块，允许模型通过空间模式匹配而非因果推理来完成预测任务*。
例如，如果掩蔽了视频末尾的一致patch 和视频开头附近的另一个patch，模型可以通过「这两个 patch 看起来相似」来猜测它们的嵌入应该接近——这是一种空间模式匹配，而非真正的时序因果推理。
seq-JEPA 的解决方案是**顺序预测**的1. 首先掩蔽时间最接近当前的未来块，预测它们的嵌入
2. 将预测的嵌入作为新的 context 追加到序列中
3. 基于扩展后的 context，继续预测下一批未来块
4. 重复直到所有掩蔽块被预测完。
这种设计迫使模型必须进行「如果A 发生，那么B 会发生」式的因果推理——因为每次预测都依赖于前序预测的结果。实验表明，seq-JEPA 在需要精确因果理解的任务（如碰撞预测、物体交互推理）上显著优化V-JEPA。
seq-JEPA 还引入了**外部记忆模块**：一个可读写的 memory bank，存储长期历史中的关键世界状态。这种seq-JEPA 能够推理远超训练窗口长度的世界变化——例如，记住 30 秒前藏起来的物体位置，并在当前帧中推理它的状态。

## JEPA 家族的贡献总结与未来方。
JEPA 家族从无到有地建立了一整套「在嵌入空间中学习世界模型」的方法论。其核心贡献可归纳为。
1. **证明了表示空间预测优于像素空间重建*：在计算效率、表示质量、下游任务适配性上都有优势
2. **解耦了视觉感知与世界动力学**：尤其是 DINO-WM 的工作，为「世界模型= 预训练视频+ 轻量动力学」的配方提供了实证支持3. **揭示了顺序预测的因果价代*：seq-JEPA 表明，预测的顺序结构对模型学到因果理解至关重。
未来方向包括中- **JEPA + State Transition 的深度融合*：在 JEPA 表示空间中嵌入更强大的动力学模型（如 RSSM的- **可解码的 JEPA**：在保持嵌入预测优势的同时，训练一个轻量解码器实现按需的视觉输入- **多模型JEPA**：将 JEPA 框架扩展到文本、音频、触觉等模态，实现跨模态的世界理解
