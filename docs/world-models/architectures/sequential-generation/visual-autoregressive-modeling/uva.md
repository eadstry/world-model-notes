## 学习笔记

### 核心思想

UVA（Unified Video Action Model）是斯坦福大学提出的统一视频生成和动作预测框架。核心理念是：视频和动作是同一物理过程的两种互补视角——视频展示发生了什么，动作为什么会发生提供了因果信号。将二者统一建模，模型不仅能更准确地预测动作，还能生成可解释的视频轨迹。

UVA 通过联合视频-动作潜空间和分支式生成头实现了统一框架。模型在训练时学习视频 token 和动作 token 的共享表示；运行时可根据需求切换生成模式：纯动作生成（用于机器人学习）、纯视频生成（用于模拟）、或同时生成两者（用于可解释行为）。通过灵活的推理策略，单一模型能够覆盖策略学习、视频预测、模型学习和观察学习等多种任务。

### 架构设计

- **Vision Encoding（视觉编码）**：UVA 使用 VQ-VAE 将视频帧压缩为离散 token 序列。视频 token 与动作 token 通过专门的编码器分别编码后注入到联合潜空间中，该潜空间通过对比学习确保视觉和动作模态的对齐。在潜空间上，使用同一个潜变量同时监督视频重建和动作预测两个任务目标。
- **Knowledge Learning（知识学习）**：学习过程采用联合训练策略。在策略学习任务中，模型基于视频帧序列执行动作生成；在模型学习任务中，模型执行视频帧预测，学习环境动力学；在观察学习中，模型从纯视频观察中学习富语境表示。这种多任务联合训练使模型能够从不同角度理解视频-动作关系。
- **Controllable Simulation（可控模拟）**：UVA 的推理过程极为灵活。在闭环控制场景中，模型以当前视频帧为输入，直接通过动作生成头产生动作 token，无需生成中间视频帧，实现了极低的动作推理延迟（约10ms）。在可视化场景中，模型可同时生成视频和动作，提供可解释的决策轨迹。

### 关键创新

- **首次提出统一视频生成与动作预测**：通过联合潜空间实现视觉和动作的深度融合，而非简单的多任务训练。
- **分支式生成实现极速动作推理**：推理时完全绕过视频生成，仅通过动作生成头输出动作，实现机器人学习的极致效率。
- **多任务联合训练**：单一模型覆盖策略/模型/观察学习等多种学习范式。
- **通用机器人解决方案**：在策略学习、模型学习、视频预测等多种任务上均达到超越专用方法的水平。

### 实验结果要点

在多个机器人操作基准上，UVA 在动作预测精度和推理速度方面均达到与专用方法（如 Diffusion Policy）相当的水平，同时额外提供了视频预测和可视化能力。视频生成质量方面，在 FVD 等指标上也表现优异。在推理延迟方面，纯动作模式下延迟仅约 10ms，适合实时闭环控制。在未见环境/物体等泛化测试中，UVA 展现出比纯动作方法更好的泛化能力，视频分支提供了额外的自监督信号。
