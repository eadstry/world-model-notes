# Visual Autoregressive Modeling - 视觉自回归建模
::: tip 范式定义
Visual Autoregressive Modeling 是 Sequential Generation 范式的**纯视觉路线**。它将被世界模型视为一个纯粹的视觉序列生成问题，输入是历史视频帧和动作 token，输出是未来视频帧。核心思想是通过自回归的方式逐帧预测未来，实现从视频到视频的端到端世界建模。
:::
