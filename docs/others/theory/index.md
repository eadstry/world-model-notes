# Theory

理论基础研究（Theory）是世界模型领域不可或缺的思维支柱。当工程实践以前所未有的速度推动模型规模和生成质量提升时，理论工作则负责追问那些更根本的问题：世界模型的本质是什么？"理解世界"意味着什么？当前的评测方法是否真正测量了该测量的东西？本分类收录了聚焦世界模型基础理论、概念框架和方法论反思的研究工作。

理论工作的一个核心议题是**世界模型的定义与本质**。"What Does It Mean for a Neural Network to Have a World Model?" 从神经网络的内部表征出发，尝试界定"拥有世界模型"的操作性含义。"Research on World Models Is Not Merely Imitation" 则批判性地指出，简单模仿训练数据分布并不能等同于真正的世界理解。"Beyond World Models: Rethinking Understanding" 更进一步，主张将讨论从"世界模型"扩展到"世界理解"的更广阔哲学视野中。

**物理可解释性**是理论工作的另一个核心维度。"Four Principles for Physically Interpretable World Models" 提出了物理可解释世界模型的四项基本原则，为模型设计和评测提供了概念性的指导框架。"World Models Should Prioritize the Unification of Sensing and Acting" 则论证了感知与行动统一在世界模型构建中的优先地位。PRISM-WM 等工作尝试从物理可实现性的角度定义世界模型的能力边界。

**从语言到世界的桥梁**是近年来的一个新兴理论议题。"From Word to World: Can Large Language Models Build World Models?" 探讨了大语言模型是否以及在何种程度上能够从纯文本训练中涌现出世界模型能力。"Cambrian-1" 等工作从更广泛的视角讨论了多模态基础模型与世界建模之间的关系。"What Has a Foundation Model Found?" 通过实证分析探索了基础模型在训练过程中学习的内部世界表征。

**评测方法论反思**构成了理论工作的实践转向。"How Far Is Video Generation from World Model?" 和 "Closing the Train-Test Gap in World Models" 等工作系统性地审视了当前世界模型评测范式的局限性和改进方向。"AutumnBench" 等基准则从评测设计的角度提供了方法论的创新。"What Drives Success in Physical Planning?" 则通过消融实验揭示了物理规划中真正起作用的关键因素。

**认知科学与世界模型的交汇**是该分类中一个独特而深刻的子方向。"Position: Interactive Generative Video as World Model" 从交互性视角重新审视了生成式视频与世界模型的关系。"UniViD" 等工作探索了统一视觉理解与世界建模的理论可能性。EB-JEPA 等源自 JEPA 架构的工作则体现了对预测式世界模型的深层理论思考。

理论工作的研究节奏通常慢于工程实践，但其影响力更为深远。一个优秀的理论框架可以在相当长的时间内为整个领域的研究提供概念锚点和方向指引。正如 Marr 的三层次分析框架在计算神经科学中发挥了数十年的基础作用一样，世界模型领域的理论工作也在朝着构建属于自己的基础分析框架的方向努力。

本分类收录的理论工作虽然主题多样，但它们共同服务一个目标：帮助研究者从更深层次理解世界模型技术的本质、边界和发展方向。我们鼓励读者将这些理论工作与模型设计、评测基准等实践层面的内容对照阅读，以获得对该领域更为立体和全面的认知。
