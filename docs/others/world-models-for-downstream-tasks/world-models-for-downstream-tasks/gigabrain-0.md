## 学习笔记

### 核心思想

该工作提出了 GigaBrain-0，一个由世界模型驱动的视觉语言动作模型。其核心设计理念是将世界模型作为 VLA（Vision-Language-Action）系统的"内部大脑"，负责预测动作后果、规划动作序列和进行反事实推理。

该工作展示了世界模型与 VLA 架构深度融合的实践方案，为构建能够进行"深思熟虑"而非简单反应式的具身智能体提供了重要的技术参考。
