# World Models for Downstream Tasks

世界模型的下游应用（World Models for Downstream Tasks）是验证技术价值的最直接战场。如果说基础评测测量了模型的"能力"，那么下游应用则测量了模型的"效用"——即世界模型在实际场景中能够产生多大程度的积极影响。本分类专门收录展示世界模型技术如何赋能各类下游任务的研究工作。

机器人操控（Robot Manipulation）是当前世界模型下游应用最活跃的领域。PWM（Policy via World Model）和 WMPO 等基础性框架为世界模型驱动的机器人策略学习奠定了基础。DreamZero 利用世界模型实现了零样本机器人操作策略的学习，DexWM 将灵巧手操控与世界模型预测相结合，Act2Goal 通过世界模型对动作序列的后果进行推理来决定最优操作序列。ReconDreamer 和 DriveDreamer4D 则从4D世界建模的角度为机器人提供了更丰富的感知和规划能力。

自动驾驶（Autonomous Driving）是另一个关键的下游应用方向。DriveVLA 将视觉语言动作模型与世界模型融合用于端到端驾驶，Cosmos Drive Dreams 探索了大规模视频生成模型在驾驶决策中的应用。WorldEnv 构建了一个基于世界模型的驾驶环境，为强化学习驾驶策略的训练和评测提供了新方案。CoWVLA 等工作通过世界模型增强了视觉语言模型在驾驶场景中的动作预测能力。

视觉与控制策略的交叉应用涵盖了更广泛的场景。GVF-Tape 利用世界模型进行基于视觉的机器人操控策略评估，Scalable Policy Evaluation with Video World Models 提出了利用视频世界模型进行可扩展策略评估的方法。WorldLM 将世界建模与大语言模型相结合增强了场景理解能力。RIGViD 等工作室探索了基于世界模型的视频扩散策略在机器人学习中的应用。

智能体与世界模型的协同是另一个值得关注的方向。"General Agents Contain World Models" 论证了通用智能体天然包含世界模型能力的观点，这一思想对于理解世界模型在通用人工智能中的角色具有重要意义。GigaBrain-0 通过世界模型为智能体提供大规模环境交互的数据引擎，MindJourney 则从认知科学的角度探索了世界模型驱动的智能体思维推演。

异常检测与安全性评估是下游应用中较为独特的一个方向。"World Models for Anomaly Detection During Robotic Manipulation" 展示了世界模型在检测机器人操作异常中的独特价值——因为世界模型能够预测"正常情况下应该发生什么"，因此任何与预测显著偏差的观测都可能指向异常。这种基于预测的异常检测方式具有无需异常样本训练的天然优势。

仿真与合成数据生成是世界模型下游应用的另一个重要方向。WorldGym 利用世界模型构建了用于策略训练的虚拟环境，JOWA 通过世界模型实现了跨具身的数据增强。这些工作展示了世界模型在降低机器人学习数据成本方面的巨大潜力——通过在虚拟世界中进行大规模探索和训练，然后将学习成果迁移到真实世界。

从方法论的角度来看，世界模型赋能下游任务的方式可以归纳为三种基本范式：**基于模型的规划**（通过世界模型"想象"动作的后果，然后选择最优动作）、**数据增强与仿真**（利用世界模型生成合成数据来扩充训练集）和**不确定性量化**（利用世界模型的预测不确定性来引导安全探索和风险规避）。这三种范式并不是互斥的，许多最先进的下游应用系统同时集成了多种范式。

总的来说，世界模型的下游应用正在经历从"概念验证"向"实际部署"的转型。随着世界模型技术的日益成熟，其在机器人、自动驾驶、工业AI等领域的实际影响力将持续扩大。本分类将持续追踪新的应用场景和落地案例，为研究者提供从实验室到实际应用的完整技术路线图。
