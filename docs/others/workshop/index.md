# Workshop

学术研讨会（Workshop）是顶级学术会议的重要组成部分，也是世界模型研究社区开展深度交流、凝聚研究共识的关键平台。与正式的主会论文不同，Workshop 为研究者提供了一个更加自由、开放和前瞻性的讨论空间——在这里，创新的想法可以被自由分享，尚未成熟的研究方向可以被充分辩论，跨学科的视角可以发生碰撞与融合。

近年来，世界模型已成为多个顶级会议 Workshop 的核心议题。**ICLR 2025** 的 "World Models: Understanding, Modelling, and Scaling" 是该领域最具影响力的 Workshop 之一，其议题全面覆盖了世界模型的理论理解、模型设计和规模化训练三个核心维度，体现了当前社区对世界模型研究的系统性思考。该 Workshop 吸引了来自认知科学、机器学习、计算机视觉和机器人学等多个领域的研究者共同参与。

**CVPR 2025** 的 "Benchmarking World Models" Workshop 则将焦点放在了一个长期被低估但至关重要的问题上：如何科学地评测世界模型？这个 Workshop 汇聚了评测基准的设计者、视频生成模型的研究者和下游应用开发者，共同探讨评测标准的确立、评测基准的构建和评测方法的改进。这次 Workshop 的一个重要成果是推动了 WorldModelBench 等评测基准的社区标准化进程。

**ICCV 2025** 和 **ICML 2025** 也都设立了世界模型相关的 Workshop。ICCV Workshop 更多地从视觉理解的角度探讨世界模型的建模能力和评估标准，而 ICML Workshop 则更加强调世界模型与强化学习、因果推断等学习理论问题的交叉融合。这种在多会议上并行开设 Workshop 的现象，恰恰反映了世界模型研究的跨学科特性——它同时是视觉、学习和交互的问题。

**NeurIPS 2025** 的 World Models Workshop 延续了该会议对基础学习问题的关注传统，其核心议题围绕世界模型的泛化理论、不确定性量化以及安全可靠的世界建模等深层问题展开。NeurIPS Workshop 通常更加注重理论深度和方法论创新，为该领域的基础理论研究提供了重要的交流阵地。

**ICLR 2026** 的 Workshop 预告显示了该领域几个正在兴起的新方向，包括世界模型与多模态大语言模型的深度融合、世界模型在科学发现（如蛋白质折叠、天气预测）中的应用拓展，以及神经符号世界建模等交叉方向。这些前瞻性的议题预示着世界模型研究的下一个发展浪潮。

Workshop 的核心价值不仅在于学术成果的展示，更在于其为研究社区创造的非正式交流机会。在 Workshop 的圆桌讨论、海报展示和自由问答环节中，研究者可以就具体的技术细节进行深入探讨，也可以就领域的发展方向进行开放性的 brainstorm。这种面对面的学术交流对于形成领域共识、建立合作网络具有不可替代的价值。

本分类将追踪收录各顶级会议中与世界模型相关的 Workshop 信息。我们建议研究者关注的不只是已经发表的 Workshop 论文，还包括 Workshop 的组织结构、邀请报告人的研究方向以及讨论环节中涌现的前沿话题——这些信息往往能够比正式论文更早地反映领域的研究动向。
