## 学习笔记

### 核心思想

该仓库作为"面向具身智能的世界模型综述"的配套资源，系统地汇集了世界模型在机器人操控、环境交互和物理推理等方面的关键文献。与纯文本综述不同，该 GitHub 仓库为综述中引用的每一篇论文提供了可点击的链接和代码仓库地址，使综述的可复现性和实用性大大增强。

该资源采用了"综述论文+开源仓库"的双轨模式，这一模式正在成为学术界知识传播的新范式——综述论文负责系统化的概念梳理和批判性分析，开源仓库则负责提供可操作、可检索的实践资源。
