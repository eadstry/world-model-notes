## 学习笔记

### 核心思想

该仓库是清华大学团队维护的世界模型综合调研资源的配套代码库，其论文标题"理解世界还是预测未来"精准地概括了世界模型研究的核心张力——模型究竟是在进行深层的世界理解，还是仅仅在进行表面的未来预测？该仓库围绕这一核心问题，组织了系统性的文献资源和代码实现。

该仓库的一个突出特色是在综述的框架下引入了哲学性的反思，不仅关注技术细节，更关注世界模型的本质问题和评测标准。这种"技术+元思考"的组织方式使其在众多资源列表中独树一帜。
