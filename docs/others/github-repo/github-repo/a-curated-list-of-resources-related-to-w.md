## 学习笔记

### 核心思想

该仓库是专注于自动驾驶世界模型（WMAD）的资源汇总列表，系统性地整理了自动驾驶领域中与世界建模相关的数据集、模型实现和评测工具。该资源列表以自动驾驶场景为锚点，对世界模型在感知、预测、规划和仿真等环节中的应用进行了分类梳理。

对于从事自动驾驶世界模型研究的学者和工程师而言，该仓库提供了一个一站式的文献导航入口。其分类体系覆盖了从基础驾驶数据集到前沿驾驶世界模型构架的完整技术链条，并且持续跟踪最新的研究进展。
