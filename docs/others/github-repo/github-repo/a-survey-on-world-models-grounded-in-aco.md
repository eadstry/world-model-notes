## 学习笔记

### 核心思想

该工作聚焦于一个被大多数世界模型研究所忽略的维度：声音物理信息。当前世界模型研究以视觉为中心，但真实世界的物理理解需要多感官融合——声音（如碰撞声、摩擦声、环境音）包含了丰富的物理属性信息。该综述系统地梳理了基于声音物理信息的世界模型相关研究。

该资源列表从声学物理建模、视听跨模态学习和声音驱动的物理推理等角度出发，展示了声音作为物理世界理解补充模态的巨大潜力。这一独特视角为多模态世界模型的研究提供了新的生长点。
