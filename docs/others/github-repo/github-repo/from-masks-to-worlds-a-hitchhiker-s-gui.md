## 学习笔记

### 核心思想

该仓库以"从掩码到世界"为题，提供了一份面向世界模型开发者的实践指南。其组织方式类似于一部技术旅程的导航手册，从最基础的自监督掩码建模（Masked Modeling）出发，逐步引导读者进入复杂的完整世界建模（World Modeling），因此作者戏称为"搭便车指南"（Hitchhiker's Guide）。

该指南的独特之处在于它不只是简单的论文列表，而是按照技术难度和概念层级组织了一条递进式的学习路径。对于希望从零开始系统学习世界模型技术的研究者和开发者来说，这是一个极具实践指导价值的入门资源。
