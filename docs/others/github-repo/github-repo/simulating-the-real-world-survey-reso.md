## 学习笔记

### 核心思想

该仓库围绕"真实世界仿真"这一宏大主题，汇集了世界模拟器相关的综述文献和技术资源。与专注于特定模型架构或应用领域的资源列表不同，该仓库从"模拟真实世界"的系统工程视角出发，关注物理仿真、视觉渲染、传感器建模和多智能体交互等多个关键技术环节。

对于从事世界模拟器开发的研究者和工程师来说，该仓库提供了一个全栈式的技术资源汇总——不仅包括学术论文，还涵盖了开源仿真引擎、3D资产库和物理引擎等技术组件，具有很强的工程参考价值。
