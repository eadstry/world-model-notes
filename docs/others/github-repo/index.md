# GitHub Repo

开源代码仓库和资源列表（GitHub Repos & Awesome Lists）是世界模型研究生态的重要组成部分。如果说学术论文提供了思想和算法，那么开源仓库则将这些思想转化为了可运行、可复现、可扩展的工程实践。本分类系统收录了与世界模型相关的高质量 GitHub 资源，涵盖模型实现、论文列表整理、工具集和教程等多个类别。

最典型的一类资源是 **Awesome 论文列表**。这些由社区维护的资源列表以精心筛选和分类的方式整理了领域内的关键论文。"A Curated List of Amazing Works in World Modeling" 以最广泛的视角覆盖了具身智能、自动驾驶、自然语言处理等多个应用领域的世界模型研究。"Awesome 3D and 4D World Models" 专注于三维和四维世界建模。"Awesome from Video Generation to World Models" 则追踪了从视频生成技术向世界模型演进的技术脉络。"A Curated List of Resources Related to World Models" 提供了更为全面的资源整理，包括数据集、教程和博客文章。

**领域特定的资源列表** 为特定研究方向提供了更聚焦的导航。"A Curated List of World Models for Autonomous Driving" 专门整理了自动驾驶领域的世界模型资源，"World Models for Autonomous Driving and Simulation" 则从仿真与建模的双重视角进行整理。"A Survey on World Models Grounded in Action Observation" 整理了基于动作观察的世界模型相关文献。

**工具和教程类资源** 也占据了重要位置。"From Masks to Worlds: A Hitchhiker's Guide to World Model Development" 提供了一篇面向开发者的实践指南。"Simulating the Real World: Survey & Resources" 汇集了世界模拟器的综述和相关资源。"Understanding World or Predicting Future? A Comprehensive Survey & Repository" 则围绕"理解世界还是预测未来"这一核心议题组织了针对性的资源列表。

开源仓库的价值不仅体现在知识整理层面，更体现在其对研究可复现性的推动上。附带代码实现的资源列表（如 "A Comprehensive Survey on World Models"）能够帮助研究者快速复现和比较不同方法的性能，从而加速领域内的技术迭代。此外，维护良好的 Awesome 列表通常会标注论文的发表年份和引用量，为研究者提供了衡量论文影响力的辅助参考。

我们鼓励研究者在参考本分类中的资源时，积极向社区贡献更新。世界模型领域发展迅速，开源仓库的资源列表需要持续的社区维护才能保持其时效性和参考价值。通过分享新发现的优秀仓库、修正过时的链接、补充遗漏的重要论文，每个人都可以为这一知识生态的健康发展做出贡献。
