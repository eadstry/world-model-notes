# Others

在世界模型（World Model）的研究生态中，除了核心的模型设计和数据集评测之外，还存在大量不可或缺的"周边"资源——综述论文、开源代码库、学术研讨会、理论基础研究、下游应用探索以及非主流视角的思考。这些资源共同构成了世界模型研究的完整知识图谱，本分类（Others）专门收纳这些跨领域、跨视角的补充性材料。

本分类下设六个子方向，各自承担着不同的知识整合与传播功能。**综述论文（Survey）** 方向汇集了该领域内最具代表性的综述文献，为研究者提供系统化的领域概览和前沿追踪服务。从世界模型在具身智能中的应用到自动驾驶领域的专项综述，从视频生成模型作为世界模拟器的可行性讨论到世界模型的安全挑战，这些综述覆盖了该领域的几乎所有重要议题。

**GitHub 开源仓库（GitHub Repo）** 方向收录了与世界模型相关的高质量开源项目和技术资源列表。这些资源不仅包括模型实现的代码仓库，也包括经过社区精心整理的论文列表（Awesome Lists）、教程和工具集。对于刚进入该领域的研究者而言，这些整理好的资源列表是最佳的学习起点；对于资深研究者而言，它们则是跟踪最新进展的有效工具。

**学术研讨会（Workshop）** 方向追踪了顶级学术会议上世界模型相关的专题研讨会。近年来，NeurIPS、ICLR、CVPR、ICML、ICCV 等顶会均设立了世界模型相关的 workshop，如 ICLR 2025 的"World Models: Understanding, Modelling, and Scaling"和 CVPR 2025 的"Benchmarking World Models"。这些研讨会不仅是学术成果的展示平台，更是领域共识形成和方向讨论的核心场所。

**理论基础（Theory）** 方向收录了聚焦世界模型理论基础和概念框架的研究论文。这些工作不直接提出新的模型架构，而是深入探讨世界模型的本质性问题：什么是世界模型？如何定义模型的"理解"？物理可解释性和因果推断在世界模型中的角色是什么？这些理论性的反思对于该领域的长期健康发展至关重要。

**世界模型的下游应用（World Models for Downstream Tasks）** 方向展示了世界模型技术在实际场景中的各种落地方案。从机器人操控到自动驾驶，从异常检测到药物研发，世界模型正在越来越多的下游任务中展现出其独特价值。这些应用案例不仅验证了世界模型技术的实用性，也为技术改进提供了来自真实世界的反馈信号。

**其他世界建模视角（Other Perspectives of World Modeling）** 方向收纳了那些不在主流技术路线之中、但具有独特思想贡献的工作。这些工作可能从不同的理论框架（如认知科学、神经科学）出发来理解世界模型，可能提出了挑战现有范式的新概念，也可能尝试将世界模型与看似不相关的领域建立联系。正是这些不同视角的碰撞与融合，推动着世界模型研究不断突破既有边界。

Others 分类体现了本学习笔记库"广纳百川"的定位理念。世界模型是一个高度跨学科的研究领域，其发展不仅依赖核心技术的突破，也依赖于理论反思、社区建设、资源整合和多视角碰撞的共同推动。我们相信，通过系统化地整理这些"周边"资源，能够为研究者提供一个更加完整的思维工具箱，帮助大家在世界模型研究的探索之路上走得更深更远。
