# 📖 关于本笔记库

本笔记库基于综述论文 **"From Seeing to Knowing the World: A Survey of Vision World Models"** 整理而成，旨在系统地梳理视觉世界模型（Vision World Models）领域的核心知识。

> 论文原始仓库：[Awesome-Vision-World-Model](https://github.com/AIWorldLab/Awesome-Vision-World-Model)

## 🏗️ 分类体系概览

综述将视觉世界模型按**架构范式**划分为四大类别：

1. **基于视觉语言模型 (VLM-based)** — 将世界建模融入多模态大语言模型
2. **基于生成模型 (Generative-based)** — 以扩散/自回归模型为核心构建世界表征
3. **基于预测模型 (Prediction-based)** — 通过未来帧/状态预测学习世界动态
4. **混合架构 (Hybrid)** — 融合多种范式的综合方案
